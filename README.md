# QuickDig UI

A professional, lightweight, and login-free web interface to monitor game server status using GameDig.

## Overview
QuickDig UI allows users to instantly query any supported game server (e.g., Project Reality, Battlefield 2, Arma, CS:GO) by simply entering the server IP and Port. The tool automatically fetches real-time data and displays it in a clean, split-team interface.

## Key Features
* **Split Team Roster**: Automatically divides players into their respective teams or factions (e.g., US vs MEC).
* **Live Silent Updates**: The server data and player list automatically refresh every 10 seconds in the background.
* **Detailed Server Info**: Displays current map, game mode (e.g., AAS, CQ), and live player count.
* **Zero Configuration**: Users do not need to download software, log in, or configure settings.

## Supported Games
This project is powered by the Node.js GameDig library, meaning it inherently supports over 300 game titles. While the default query type is set to Battlefield 2 / Project Reality, it can seamlessly work with any game protocol supported by GameDig.
